# 긴 컨텐츠 / 스크롤 처리

현재 구현된 기본 흐름:
- `추가 읽기` 액션
- `세션 초기화` 액션
- 스크롤 후 새 OCR 결과를 누적 세션에 추가
- 이전 세션 말미와 새 스냅샷 시작 부분을 비교한 중복 줄 제거
- `현재 화면 기준 도움`과 `누적 세션 도움` 분리

메모:
- 자동 스크롤은 아직 의도적으로 넣지 않았다.
- 앱별 스크롤 동작, sticky header, lazy loading, 광고, 포커스 이동 같은 변수가 커서 수동 스크롤 + 누적 세션이 더 안정적이다.

다음에 검토할 수 있는 확장:
- sticky header 같은 반복 노이즈 제거 규칙 강화
- 누적 세션 내 검색/하이라이트
- 누적 세션 내보내기 또는 저장
